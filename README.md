# アプリ説明
このアプリケーションは、サーバーとクライアントで動作します。シンプルなチャットアプリです。あまり多くチャットを送りすぎるとRAMを圧迫しますので、お気を付けください。なお、このRAM圧迫問題は1.0.3で解消する予定です。また、今後はファイルや画像を送信できるようにしたいと思っていますが、できる限り軽量な形を保持したいため、どうなるかは知りません。

distの中のexeファイルでやってください。
